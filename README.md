**Syncify**


Syncify is a web-based multiuser calendar system that allows multiple users to manage and share their appointments.
Each user can create, view, update, and delete personal events, as well as schedule multi-occurrence events such as daily or weekly meetings. 
Supports shared events, where an event created by one user can be shared with other users, and any update made by the creator is automatically reflected for all participants. 
Syncify application provides an easy-to-use web interface for scheduling events, inviting users, and viewing calendars in a structured manner.

This project incorporates a chatbot agent which helps us check the slot availability among other participants of the event and schedule an event automatically.
Syncify also allows users to schedule email reminders to their inbox before the event.
